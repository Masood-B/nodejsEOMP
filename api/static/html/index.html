Route Description

User Route

HTTP Methods Path Description
GET /users Get all the users
GET /user/:id Get a single user
POST /user register a user
POST /login Clerify if the user is on the system
PATCH/PUT /user/:id Update the user's information
DELETE /user/:id Remove the user from the system

Product Route

HTTP Methods Path Description
GET /products Get all the products
GET /product/:id Get a single product
POST /product register a product
PATCH/PUT /product/:id Update the product's information
DELETE /product/:id Remove the product from the system
