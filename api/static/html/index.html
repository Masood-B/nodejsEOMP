Route Description

User Route

HTTP Methods Path Description
GET /users Get all the users
GET /users/:id Get a single user
POST /users register a user
POST /users/ Clerify if the user is on the system
PATCH/PUT /users/:id Update the user's information
DELETE /users/:id Remove the user from the system

Product Route

HTTP Methods Path Description
GET /products Get all the products
GET /products/:id Get a single product
GET products/name/:id Get a product's name (Hint: Use Letters)
GET /products/amount/asc Ascending order of the amount
GET /products/amount/desc Descending order of the amount
GET /products/name/asc Ascending order of names
GET /products/name/desc Descending order of names
GET products/name/:id Get a product's name (Hint: Use Letters)
GET /products/thriller Get thriller category of movies
GET /products/drama Get drama category of movies
GET /products/comedy Get comedy category of movies
GET /products/fantasy Get fantasy category of movies
GET /products/musical Get musical category of movies
GET /products/mystery Get mystery category of movies
GET /products/zombie Get zombie category of movies
POST /products register a product
PATCH/PUT /products/:id Update the product's information
DELETE /product/:id Remove the product from the system
